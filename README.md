# Adaptive Fitness App

Adaptieve, uitlegbare trainingsschema's. Zie [PROJECT.md](./PROJECT.md) voor
het plan van aanpak, architectuurkeuzes en gemaakte aannames voor Fase 1.

## Quickstart

```bash
npm install
cp .env.example .env   # vul je Supabase-projectgegevens in
npm run test           # progressie-engine unit tests
npm run typecheck
npm run web
```
